# User Manual

## Introduction
This web system is designed to provide users with easy access to basic school-related information through a web interface.

This manual explains how to register, log in, use the dashboard and log out of the system.

## System Access
To access the system, open the 'login.html' file using any modern web browser such as Google Chrome or Mozilla Firefox.

## Creating an Account (Registration)
1. On the login page, click **"Register here"**
2. The Registration page will open
3. Enter the required information:
 - Full Name
 - Username
 - Email Address
 - Password
 - Confirm Password
4. Click the **Register** button
5. After succesful registration, you will be redirected to the login page

## Login Out
1. On the Dashboard page, click the **Logout** link
2. You will be redirected back to the Login page
3. Your session will end
