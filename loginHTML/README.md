## System Overview
The web system is a simple web-based application designed to manage basic student access to a school portal.

The system allows users to:
- Register an account
- Log in to the system
- View a dashboard with student information
- Log out securely

This system is built using HTML and CSS and focuses on front-end design, navigation and usability.

## System Features
- User registdration page for new users
- Login page for registered users
- Dahboard page displayin student information
- Consistent design across all pages
- Simple and clear page navigation using links
- User-friendly interface

## File Structure
- login.html - user login interface
- register.html - user account registration page
- dashboard.html - user dashboard with student data
- styles.css - system-wide design and layout
- USER_MANUAL.md - guide for system users
- README.md - sytem documentation

## System Limitaitions
- The system does not stor user data permanently
- No backened or databse is implemented
- All data shown in the dashboard is sample data
